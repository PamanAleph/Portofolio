## Overview

President’s Pick-Up is a **package delivery system** designed to facilitate shipping between designated dormitories. Built with **PHP Native, MySQL, and TailwindCSS**, the platform provides users with a seamless way to send and track shipments while allowing administrators and couriers to manage deliveries efficiently.

## Key Features

- **Authentication System**: Secure login system for **admins and couriers** to manage deliveries.
- **Order Management**: Users can create shipping requests, providing sender and recipient details.
- **Price Calculation**: Shipping costs are determined based on **location, service type, weight, and insurance**.
- **Package Tracking**: Real-time status updates (Processing, Lost, Delivered) managed by admins and couriers.
- **Courier Management**: Admins can assign couriers to specific destinations and validate deliveries.
- **Dashboard & Analytics**: Data visualization on order counts, delivery success rates, and shipment statuses.

## Technologies Used

- **PHP Native** – Backend logic for handling order processing and user authentication.
- **MySQL** – Structured database design ensuring efficient data storage.
- **TailwindCSS** – Responsive and modern UI styling for a clean user experience.
- **JavaScript & AJAX** – Enhancing real-time package tracking.

## Challenges & Solutions

### 1. **Ensuring Secure User Authentication**
**Challenge:** Unauthorized access to courier/admin pages.
**Solution:** Implemented **secure session handling** and role-based access controls.

### 2. **Accurate Price Calculation**
**Challenge:** Dynamic shipping cost calculations based on various parameters.
**Solution:** Created an algorithm factoring in **location, service type, weight, and insurance**.

### 3. **Efficient Order & Courier Management**
**Challenge:** Assigning couriers dynamically while preventing data redundancy.
**Solution:** Implemented **randomized courier assignment** based on delivery zones.

## Outcome

By leveraging **PHP Native, MySQL, and TailwindCSS**, President’s Pick-Up provides a **fully functional, efficient, and user-friendly package delivery system**. The system ensures **smooth shipment processing, real-time tracking, and secure admin/courier management**, making it a practical solution for intra-campus deliveries.
